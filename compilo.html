Why Compilo
Everything you need to compile with confidence

Built for teams who need to organize, consolidate, and distribute documents at scale — without the manual work.

Smart filters
Filter by filename prefix, content keywords, file type, or date. Preview matched files before you run.
Beautiful styles
Choose from curated themes — soft professional, bold editorial, clean minimal — or build your own.
Book mode
Auto-generate a title page and clickable table of contents. Every section begins with a styled chapter header.
Mixed file types
PDFs, Word docs, PowerPoint, and text files — all in one compilation. Output as PDF or Word.
Runs on a schedule

Organized outputs

How it works
Four steps. One polished document.

Compilo lives in your Google Drive sidebar. No new tab, no new tool to learn.

1
Choose your source
Pick any Google Drive folder. Select subfolders to include — or go fully recursive.
2
Set your filters
Filter by filename, keywords, or file type. See matched files in real time before you run.
3
Choose a style
Pick a theme, toggle book mode, configure your cover page and table of contents.
4
Run & schedule
Run now or schedule monthly. Files land in your output folder automatically — dated and organized.

FAQ
Questions we hear a lot
Compilo ingests PDFs, Microsoft Word (.docx), PowerPoint (.pptx), and plain text (.txt) files from Google Drive. Mixed file types in a single compilation are fully supported — Compilo handles conversion automatically. Output can be a merged PDF or a Word document.
Filter by filename prefix, contains, or ends with. You can also filter by keywords found inside the document content — useful when filenames aren't consistent. Multiple filters can be stacked, and you can preview which files match before running.
Book mode generates a polished title page followed by a clickable table of contents that links directly to each document's first page. Each document begins on a new page with a styled chapter header. Perfect for compilations you want to read or share as a cohesive document.
Yes. After setting up a compilation, you can schedule it to run daily, weekly, or monthly. Scheduled runs save automatically to your chosen output folder in Google Drive with a date stamp in the filename.
No. Compilo runs as a Google Apps Script add-on inside your Google Workspace environment. Your files flow directly within Google Drive — they are never uploaded to or stored on Compilo's servers.
Not at all. Compilo is a sidebar add-on that lives inside Google Drive. Install it from the Google Workspace Marketplace, open the sidebar, and follow the four-step setup. No code, no configuration files, no terminal.
